# up360AnswerAnalysis
This is about how to get  up360's(天学网) listening test's answers. This project is based on Bilibili streamer llroxy(Original link: https://www.bilibili.com/video/BV1MDTgzyEt9/?spm_id_from=333.1387.upload.video_card.click) and 茯芺苒(original link: https://www.bilibili.com/video/BV1bVGVzNEtA/?spm_id_from=333.1387.homepage.video_card.click), but it  simplifies the process of obtaining answers. The project cleans the html tags in the JS document to ensure reliability.

The project allows users quickly and accurately to get listening test's answers and shows the answers in an GUI window.
Users should first use a network packet capture tool to parse up360 and obtain the question folder. Then place question folder in the same folder as the main program.After that, run the main program and you'll see a GUI window with answers on it.
